# Static-Image-Pose-Detection-MediaPipe-OpenCV
This project performs human pose landmark detection on a static image using MediaPipe and OpenCV.

## 🚀 Features
- Detects 33 human body landmarks
- Works on static images
- Uses MediaPipe Pose model
- Visualizes keypoints and connections
- Custom drawing styles for landmarks

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Matplotlib

## 📂 How It Works
1. Load an input image
2. Convert image to RGB format
3. Run MediaPipe Pose detection
4. Draw detected landmarks
5. Display the annotated image

## 📌 Output

The script displays the image with detected pose landmarks and connections.
